About Python & Turtle

What is Python & Turtle?

Python & Turtle is a Python playground allowing to interact with the Python language within your browser. It offers three environments:

  1. A scratch pad exposing a Python editor and an output pane (text base),
  2. An environment allowing to interact with the Python Turtle module (graphical base),
  3. A classic Python interpreter (shell).
